# STM32F103x带网口原理图资源

## 资源描述

本仓库提供了一个STM32F103x带网口的原理图资源文件。该原理图详细描述了STM32F103x最小系统的构建，并通过片上外设SPI接口连接了以太网控制器ENC28J60。

## 资源内容

- **原理图文件**：包含STM32F103x最小系统的电路设计，以及通过SPI接口连接的ENC28J60以太网控制器。

1. **下载资源**：点击仓库中的资源文件进行下载。

3. **参考设计**：根据原理图进行电路设计和开发。
